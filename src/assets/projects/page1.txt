ELECTRICITY MONITORING DASHBOARDProject OverviewDeveloped an electricity monitoring dashboard for one production plant with 30 critical high-consumption machines. Thesystem enables real-time monitoring of energy usage and supports data-driven energy efficiency improvement.Dashboard page summaryThe electricity monitoring system was designed using Schneider PM2200 power meters installed across 30 critical high-energy consumption machines within the production plant. Energy data collected via Modbus RTU was converted to ModbusTCP to improve communication reliability and simplify network integration. The devices were connected to an industrial PCacting as an edge device within the plant network. An ETL process was implemented using Telegraf to acquire, process, andforward the data into both SQL Database and Time-Series Database (TSDB) for structured storage and historical analysis. Theprocessed data was then transmitted to a central server where it was integrated into a dashboard visualization system and datawarehouse environment, enabling real-time monitoring, historical trend analysis, and energy performance evaluation.Architecture Network and DevicePM 2220 in Panel Machine USR-W610 In Panel Distribution PC Industrial In Production Plant Office
