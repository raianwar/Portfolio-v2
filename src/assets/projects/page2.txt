OEE MONITORING DASHBOARDProject OverviewDesigned and implemented real-time OEE monitoring system for 52 production machines. The system enables managementto monitor downtime history, production quantity, cycle time, and machine speed.
OEE Monitoring DashboardThe OEE monitoring system was developed by integrating shopfloor-level PLCs with a centralized MES-oriented dataarchitecture. PLCs were configured to capture machine status signals, production counts, cycle time, and downtime conditions.Data was transmitted from the PLCs to a background application developed in C# using a socket-based communication protocol.The application processed machine logic, executed stored procedures, and managed data transactions within SQL Server toensure real-time and accurate data recording. The stored data was then utilized for dynamic dashboard visualization, allowingmanagement to monitor production performance, downtime history, cycle efficiency, and machine speed across 52 productionmachines. This architecture ensured seamless integration from shopfloor control systems to enterprise-level performancemonitoring.Architecture OEE Dashboard
